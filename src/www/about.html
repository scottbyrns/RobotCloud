How Good Robotics Cloud works

Good Robotics Cloud enables robot builders to cloud source mapping, weather, and recognition. Good Robotics Cloud is a data collection system that collects real time data and makes it available to a global community.

Interplanetary Exploration is an immersive social network for robots combining realtime data from sources all over the world into a central community. Users create their own robots to add into the platform, and play in gamified missions to explore the Universe in new and exciting ways.
