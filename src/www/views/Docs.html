  • /robot

    /robot/create

    To create a robot you POST your token and the robot to the end point. 

    				        params: {
				            robot: {
				                name: 'name',
				                description: 'desc',
				                avatar: '/images/robot.jpg'
				            },
				            token: '6E5A95D2-B066-D1C7-C849-1983B4ACDEA3'
				        },
				        route: {
				            path: '/robot/create',
				            method: 'POST'				        }
